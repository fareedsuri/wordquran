# Usage Guide - Word Quran Add-in

## Step-by-Step Instructions

This guide walks you through using the Word Quran add-in to create memorization tables.

### Prerequisites
Before you begin, make sure you have:
- Microsoft Word (2016 or later, Desktop or Online)
- The Word Quran add-in installed and loaded

### Step 1: Open Word Document
1. Open Microsoft Word
2. Create a new document or open an existing one where you want to add the memorization table

### Step 2: Access the Add-on
1. Look for the **HOME** tab in the Word ribbon
2. Find the **Word Quran** group/section
3. Click the **Show Taskpane** button
4. The Word Quran taskpane will open on the right side of your document

### Step 3: Select a Juz
1. In the taskpane, you'll see a dropdown menu labeled **"Select Juz:"**
2. Click the dropdown to see all 30 Juz options
3. Each option shows:
   - Juz number and name (e.g., "Juz 1")
   - Starting Surah and verse
   - Ending Surah and verse
4. Select the Juz you want to create a table for

Example selections:
- **Juz 1** (Al-Fatihah 1 - Al-Baqarah 141)
- **Juz 30** (An-Naba 1 - An-Nas 6)

### Step 4: Configure Table Options
Before creating the table, customize it with these options:

#### Include page numbers column
- ‚òë **Checked** (default): Adds a column showing estimated page ranges
- ‚òê **Unchecked**: Removes the page numbers column

#### Include date column
- ‚òë **Checked** (default): Adds a column for recording memorization dates
- ‚òê **Unchecked**: Removes the date column

#### Include notes column
- ‚òê **Unchecked** (default): No notes column
- ‚òë **Checked**: Adds a column for additional notes or observations

### Step 5: Create the Table
1. Once you've selected a Juz and configured your options, the **"Create Memorization Table"** button will become active (green)
2. Click the **"Create Memorization Table"** button
3. Wait a moment while the table is generated
4. You'll see a success message: "Table created successfully!"

### Step 6: Use the Table
The generated table will appear at the end of your document with:

#### Header Section
- **Title**: "Memorization Table - Juz X"
- **Range**: Surah and verse information
- **Formatted** with centered, bold heading style

#### Table Structure
The table includes these columns (based on your selections):

1. **Day**: Shows "Day 1", "Day 2", etc. (20 rows for a typical Juz)
2. **Pages** (optional): Estimated page range for that section
3. **Verses**: Section identifier for that day's portion
4. **Date** (optional): Empty field to fill in when you memorize
5. **Completed**: Checkbox symbol (‚òê) to mark when done
6. **Notes** (optional): Space for additional notes

#### Table Features
- Professional styling with a light grid
- Header row with white text on dark background
- Properly sized columns for easy reading
- Ready to print or use digitally

### Step 7: Fill in the Table
As you memorize:
1. Record the date when you start each section
2. Mark the checkbox when completed (replace ‚òê with ‚òë or ‚úì)
3. Add notes about difficult verses or reminders
4. Track your progress day by day

### Tips for Best Results

#### Planning Your Memorization
- Use one table per Juz
- Break each Juz into manageable daily portions
- Adjust the number of rows if needed

#### Customizing After Creation
You can manually:
- Add or remove rows as needed
- Adjust column widths
- Change colors or styling
- Add more detailed verse ranges

#### Multiple Juz
To create tables for multiple Juz:
1. Create the first table
2. Keep the taskpane open
3. Select another Juz
4. Click "Create Memorization Table" again
5. Each table will be added to your document

#### Organizing Your Document
Consider:
- Creating a table of contents
- Adding page breaks between Juz
- Including review schedules
- Adding motivational quotes or du'as

### Troubleshooting

#### Button is Disabled
- Make sure you've selected a Juz from the dropdown
- The button only activates after a selection

#### Table Not Appearing
- Check that you have permission to edit the document
- Try scrolling down - tables appear at the end
- Ensure the add-in loaded correctly

#### Incorrect Juz Information
- Each Juz is pre-loaded with accurate Surah/verse ranges
- If you notice an issue, please report it on GitHub

#### Need to Start Over
- Simply undo (Ctrl+Z) to remove the last table
- Or manually delete the table and create a new one

### Example Workflow

Here's a complete example:

1. **Goal**: Create a 20-day memorization plan for Juz 30
2. **Open** Word and the add-in
3. **Select** "Juz 30 (An-Naba 1 - An-Nas 6)"
4. **Options**: Check all three options (pages, date, notes)
5. **Create** the table
6. **Customize**: Edit the "Verses" column with specific Surah ranges:
   - Day 1: An-Naba 1-20
   - Day 2: An-Naba 21-40
   - etc.
7. **Print** or save as PDF
8. **Use** daily to track progress
